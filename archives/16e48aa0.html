Ribbon 配置自定义

配置自定义可以分为如下两种：

  1. 使用 Java 代码自定义 Ribbon 配置，配置指定名称的 Ribbon Client 实际上就是通过注解声明式的实现了 Ribbon Client 的负载均衡策略
  2. 使用属性自定义 Ribbon 配置，客观感觉此种方式也是比 Java 代码配置的方式更加方便，此章节的 demo 示例也是基于此种形式实现的。
